# 🚀 Project Atlas – Enterprise Project Management Intelligence Platform

## 📌 Overview

**Project Atlas** is an Enterprise Project Management Intelligence Platform built using **Power BI** and **Microsoft Excel** to provide end-to-end visibility into an organization's project portfolio.

The dashboard enables executives, PMOs, project managers, resource managers, and finance teams to monitor project execution, resource utilization, sprint performance, financial health, and organizational risks through interactive visual analytics.

The objective of this project is to transform raw project management data into actionable business insights using data modeling, DAX, and interactive dashboards.

---

# 🎯 Objectives

- Build a centralized PMO analytics platform
- Monitor project portfolio performance
- Track resource utilization across departments
- Analyze Agile sprint execution
- Identify project risks and critical issues
- Compare planned vs actual financial performance
- Enable interactive decision making using Power BI

# 📂 Data Model

The project follows a relational star-schema inspired model consisting of multiple interconnected tables.

### Core Tables

- Projects
- Employees
- Departments
- Clients
- Tasks
- Milestones
- Sprints
- Resources
- Budgets
- Risks
- Issues
- Timesheets

Relationships were established using Power BI's Model View to support interactive cross-filtering across dashboards.
